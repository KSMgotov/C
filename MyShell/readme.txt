Программа представляет собой упрощенную версию Shell. Она умеет читать
команды и создавать отдельные процессы для их выполнения.
Поддерживается создание конвейеров, перенаправление ввода и вывода,
запуск в фоновом режиме и еще некоторые функции. Команды хранятся в виде
специального дерева, с помощью которого сохраняется информация о самих
командах и очередности их выполнения.
